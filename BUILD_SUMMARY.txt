HOW TO RUN
================================================================================

1. INSTALLATION
   cd /sessions/amazing-lucid-brown/fred-app
   npm install

2. START DEVELOPMENT SERVER
   npx expo start --web
   
   OR for native:
   npx expo start --ios    # macOS only
   npx expo start --android

3. WEB TESTING
   - Press 'w' when expo start runs
   - Opens http://localhost:19006 in browser

4. MOBILE TESTING
   - Use Expo Go app (iOS/Android)
   - Scan QR code from terminal output

5. LOGIN WITH DEMO ACCOUNTS
